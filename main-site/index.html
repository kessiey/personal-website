Site Architecture

The site is built using microservices architecture and is hosted on a Kubernetes cluster on DigitalOcean. Each component of the site, including the blog, current site, cloud resume, and games, runs as a separate application in the cluster.

The blog's frontend is connected to a managed RDS (Relational Database Service) instance, ensuring reliable and scalable database management.

Continuous deployment is automated with GitHub Actions, streamlining updates and ensuring seamless integration of new features. The entire codebase is open-source and available on my GitHub repository.
